Railway & Test Tools

A small collection of browser-based utilities

📐

ETCS Inclinometer Helper V2

Inclinometer tool with visual radar orientation. Pitch with triangle sign selector, tilt, and track slope measurement.
🚂

UIC 7-Digit Check Calculator

Calculate and verify the UIC check digit for 7-digit rolling stock numbers (Výpočet kontrolní UIC číslice). Used in CZ, DE and others.

Experimental

🔢

UIC 12-Digit EVN Calculator

Full 12-digit European Vehicle Number calculator with type code and country code awareness. Supports tractive stock (UIC 438-3), coaches (438-1) and freight wagons (438-2).
📡

Fluke ScopeMeter Viewer

Retrieve and render waveform images from older Fluke ScopeMeters via PostScript output over optical serial cables.

Legacy

📐

ETCS Inclinometer Helper

Original version. Convert and validate inclinometer readings for ETCS track geometry assessments.
Static tools — no data leaves your browser.
